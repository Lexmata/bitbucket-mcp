Installation

Get started in minutes with your preferred package manager

Terminal 
# Using npm
npm install -g @lexmata/bitbucket-mcp

# Using pnpm
pnpm add -g @lexmata/bitbucket-mcp

# Using yarn
yarn global add @lexmata/bitbucket-mcp

Configuration

Set up authentication to connect to your Bitbucket Cloud workspace

1 Access Token (Recommended)

  1. 1. Go to Bitbucket Settings > App passwords
  2. 2. Create a new app password with required permissions
  3. 3. Set the environment variable:
export BITBUCKET_ACCESS_TOKEN="your-token"

2 OAuth 2.0 (For Apps)

  1. 1. Go to Workspace Settings > OAuth consumers
  2. 2. Create a new OAuth consumer
  3. 3. Set the environment variables:
export BITBUCKET_CLIENT_ID="your-id" export BITBUCKET_CLIENT_SECRET="your-secret"

Usage with Claude Desktop

Add the server to your Claude Desktop configuration

claude_desktop_config.json 
{
  "mcpServers": {
    "bitbucket": {
      "command": "npx",
      "args": ["@lexmata/bitbucket-mcp"],
      "env": {
        "BITBUCKET_ACCESS_TOKEN": "your-access-token"
      }
    }
  }
}
macOS ~/Library/Application Support/Claude/
Windows %APPDATA%\Claude\
Linux ~/.config/Claude/

MCP Resources

Access Bitbucket data through standardized resource URIs

Repository

bitbucket://repository/{workspace}/{repo}

Access repository metadata, settings, and configuration

Pull Request

bitbucket://pullrequest/{workspace}/{repo}/{id}

Access pull request details, status, and metadata

File

bitbucket://file/{workspace}/{repo}/{path}

Access file contents from any branch or commit
